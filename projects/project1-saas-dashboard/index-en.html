1. Project: SaaS Revenue Analysis (Tableau)

Description:

  • This project analyzed the revenue performance of a SaaS (Software as a Service) company.
  • The goal was to reveal which countries and time periods brought in the most revenue.
  • Strategic decisions were supported through analysis of user behavior and payment patterns.

Technical:

  • Using Tableau, metrics like ARPPU (Average Revenue per Paying User), total revenue, and user count were visualized by time and country.
  • Dashboards presented KPIs, trend analyses, and payment behavior patterns for data-driven decision making.

Focus Areas:

  • Revenue analysis by location
  • Time series tracking
  • ARPPU
  • Analysis of paying users

Tools Used: Tableau
